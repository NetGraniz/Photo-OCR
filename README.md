# Photo OCR

Photo OCR - статический браузерный OCR-инструмент для распознавания текста с изображений. Проект работает напрямую на GitHub Pages: только HTML, CSS и JavaScript, без backend, базы данных, авторизации, npm-сборки, платных API и API-ключей.

Распознавание выполняется локально в браузере пользователя через [Tesseract.js](https://tesseract.projectnaptha.com/). Изображения не сохраняются, не отправляются на сервер и не передаются во внешние OCR-сервисы. С CDN загружается только библиотека Tesseract.js и языковые модели.

## Что умеет проект

- Вставлять изображение через `Ctrl+V`.
- Загружать изображение через выбор файла.
- Принимать изображение через drag-and-drop.
- Показывать предпросмотр и поворачивать изображение.
- Распознавать русский, английский или русский + английский текст.
- Использовать разные режимы предварительной обработки изображения.
- Использовать разные режимы OCR для обычного текста, строки, колонок или цифр.
- Автоматически запускать OCR после добавления изображения.
- Редактировать результат вручную.
- Копировать результат одной кнопкой.
- Скачать результат в `photo-ocr-result.txt`.
- Быстро очищать только результат или всё приложение.

## Как пользоваться

1. Откройте сайт.
2. Добавьте изображение одним из способов: `Ctrl+V`, выбор файла, drag-and-drop или кнопка `Вставить из буфера`.
3. Выберите язык OCR.
4. При необходимости выберите режим обработки изображения и тип текста.
5. Нажмите `Распознать текст` или включите `Автораспознавание`.
6. Дождитесь завершения OCR.
7. Отредактируйте результат вручную, если нужно.
8. Скопируйте текст или скачайте `.txt`.

## Вставка через Ctrl+V

Скопируйте изображение или сделайте скриншот, откройте страницу и нажмите `Ctrl+V`. Если в буфере есть изображение, оно автоматически появится в области предпросмотра. Старое изображение, старый результат и прогресс будут очищены.

## Загрузка файла

Нажмите на область загрузки или используйте выбор файла. Поддерживаются обычные браузерные форматы изображений: PNG, JPEG, WebP и другие форматы, которые умеет читать браузер.

## Drag-and-drop

Перетащите изображение в область загрузки. Во время перетаскивания область подсвечивается. Если перетащить не изображение, интерфейс покажет понятную ошибку.

## Кнопка вставки из буфера

Кнопка `Вставить из буфера` использует Clipboard API. В некоторых браузерах или при запрете доступа чтение буфера может быть недоступно. В этом случае используйте `Ctrl+V` или загрузку файла.

## Языки OCR

- `Русский` - модель `rus`.
- `Английский` - модель `eng`.
- `Русский + Английский` - модель `rus+eng`.

При смене языка OCR worker переинициализируется под выбранную языковую модель при следующем распознавании.

## Режимы обработки изображения

- `Оригинал` - без дополнительной обработки.
- `Скриншот` - лёгкое увеличение маленьких изображений и умеренный контраст.
- `Документ` - grayscale, повышение контраста и мягкое выравнивание яркости.
- `Сильный контраст` - более заметное усиление контрастности.
- `Чёрно-белый` - бинаризация изображения.
- `Инверсия` - инвертирует цвета перед OCR.

Большие изображения автоматически уменьшаются до безопасного размера canvas, чтобы браузер не зависал. Если это произошло, сайт покажет предупреждение.

## Режимы текста

- `Обычный текст` - базовый режим для большинства изображений.
- `Одна строка` - полезно для коротких строк, кодов, заголовков.
- `Таблица / колонки` - пытается лучше сохранить пробелы между словами и колонками.
- `Только цифры` - ограничивает распознаваемые символы набором `0123456789.,:-+/()`.

## Работа с результатом

После OCR текст появляется в большом поле. Его можно редактировать вручную.

Доступные действия:

- `Копировать` - копирует весь текст через Clipboard API.
- `Скачать .txt` - сохраняет результат в файл `photo-ocr-result.txt`.
- `Убрать лишние переносы` - склеивает переносы строк внутри обычного текста, сохраняя абзацы.
- `Убрать двойные пробелы` - заменяет повторяющиеся пробелы на один.
- `Очистить результат` - очищает только текст, не удаляя изображение.
- `Очистить всё` - удаляет изображение, результат и прогресс.

## Публикация на GitHub Pages

1. Загрузите файлы проекта в репозиторий GitHub.
2. Откройте `Settings` -> `Pages`.
3. В разделе `Build and deployment` выберите `Deploy from a branch`.
4. Укажите ветку `main` и папку `/root`.
5. Нажмите `Save`.
6. После публикации GitHub покажет ссылку на сайт.

Для репозитория `NetGraniz/Photo-OCR` сайт обычно доступен по адресу:

```text
https://netgraniz.github.io/Photo-OCR/
```

## Ограничения Tesseract.js

- Первое распознавание может быть медленным, потому что браузер загружает OCR-движок и языковые модели.
- Качество OCR зависит от резкости, контраста, размера, наклона и шума на изображении.
- Рукописный текст, сложные таблицы, мелкий шрифт и декоративные шрифты распознаются хуже.
- Очень большие изображения могут потреблять много памяти, поэтому сайт уменьшает canvas до безопасного размера.
- Для загрузки Tesseract.js и языковых моделей нужен доступ к CDN.

## Приватность

Проект не использует backend, не отправляет изображения на сервер, не хранит историю распознаваний и не использует внешние OCR API. Вся обработка изображения и текста выполняется локально в браузере пользователя.
